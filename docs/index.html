Your choice

Pick your backend.

From fully local (zero cost, zero cloud) to ultra-fast API. You decide what runs where.

Recommended
✨
Gemini Flash
~$3/yr
Audio stays local
Best quality/cost ratio. Google's Gemini Flash is fast, smart, and nearly free.
openvoiceflow --set-key gemini YOUR_KEY
⚡
Groq
Free tier / ~$1/yr
API call
Fastest inference on the planet. Llama on Groq's custom silicon.
openvoiceflow --set-key groq YOUR_KEY
🤖
OpenAI
~$5/yr
API call
GPT-4o mini. Great for complex cleanup and nuanced rewrites.
openvoiceflow --set-key openai YOUR_KEY
🌟
Claude
~$5/yr
API call
Anthropic's Claude Haiku. Exceptional writing quality and tone.
openvoiceflow --set-key claude YOUR_KEY
100% Local
🦙
Ollama
$0 / forever
Never leaves Mac
Fully offline. Llama 3, Mistral, or any local model. Air-gapped privacy.
openvoiceflow --backend ollama
⚙️
None (Whisper only)
$0 / forever
Never leaves Mac
Raw Whisper output only. No LLM cleanup. Minimal and lightning fast.
openvoiceflow --backend none
💡 All options: Audio transcription is always local via Whisper. Only the cleanup LLM call goes to an API (if you choose one).

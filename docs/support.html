Installation, permissions, and privacy help for OpenVoiceFlow.

Start here

  • Install OpenVoiceFlow and move it to Applications.
  • Open the app from Applications and follow the in-app setup.
  • Allow Microphone access when you choose to test dictation.
  • Use the on-screen record control if a global shortcut conflicts.

Permission help

OpenVoiceFlow explains permissions in context. For the direct build, Accessibility can enable automatic paste and optional correction learning. The Store target is designed to work without broad cross-app observation.

If macOS does not list the app after a prompt, quit it completely, reopen the copy in Applications, and try the relevant in-app test again.

Report a problem

Please include your OpenVoiceFlow version, macOS version, Mac architecture, and a short description of what happened. Do not attach dictated text, audio, API keys, or private logs unless you have reviewed them.

Open a GitHub issue

Security or privacy concern

For a vulnerability or sensitive disclosure, follow the repository’s security policy. For data-handling questions, read the privacy page.

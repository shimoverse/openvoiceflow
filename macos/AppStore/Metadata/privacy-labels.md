## Data not linked to the user

## Local-only processing disclosure

Microphone audio is recorded only while the user invokes dictation, stored temporarily inside the app sandbox, and processed locally by the bundled whisper.cpp runtime. The resulting transcript remains on the Mac and is copied to the system pasteboard. OpenVoiceFlow has no analytics, account system, crash reporter, advertising SDK, cloud transcription, or runtime network client entitlement in this Store target.

These answers apply only to `OpenVoiceFlowStore` (`com.shimoverse.openvoiceflow`). Reassess them before adding cloud LLM cleanup, telemetry, update checks, support uploads, or any runtime network feature.

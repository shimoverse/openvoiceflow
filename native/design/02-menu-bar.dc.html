Ordering rationale
Status header first — the menu is a glance surface before it is a control surface. Primary action (Start/Stop) next, then the three pickers that define the pipeline, then windows, then Quit — destructive always last, behind a separator.
MenuBarExtra idioms
Menu-style (not window-style) for zero-latency open. Submenus are native Menu {} nests. SF Symbols: mic, keyboard, cpu, sparkles, gauge, gearshape, power — 13 pt, monochrome, .secondary tint.
Icon behavior
Template image, ~18 pt. Idle is static. Listening ripples ±1 px (static glyph swap under Reduce Motion). Working spins the coil. Success flashes a 600 ms fill, then returns to idle. Paused dims to 45 % with pause ticks. Error shows the broken line until acknowledged.

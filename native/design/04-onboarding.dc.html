Priming beats pleading
Each screen explains the why in our voice before the macOS dialog appears, and previews exactly what the system will show — so the scary system prompt is expected, not alarming.
The wait builds trust
The model download is the longest moment we'll ever make you wait — so it narrates what's happening locally (download → checksum → Neural Engine compile) and ends with the offline promise.
Rehearsal, then the real thing
The hotkey is rehearsed on screen 5, then used for real on screen 7 — the aha is doing the actual gesture, not reading about it. Ends pointing at the menu bar: the app's permanent home.
